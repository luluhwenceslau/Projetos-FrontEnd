HTML: Principais Tags

Logo abaixo se encontram as principais Tags HTML utilizadas em um site

  • HTML: Indica o começo da página
  • BODY: Corpo principal
  • HEADER: Cabeçalho
  • H1: Titulo Principal, variando de H1 até H6, por ordem de prioridade
  • UL: Para listas não ordenadas
  • OL: Lista ordenada
  • LI: Itens das listas
  • MAIN: Conteudo Principal
  • SECTION: Separa o conteudo em sessões
  • FOOTER: Rodapé
  • FORM: Formulários
  • BUTTON: Botões
  • P: Paragráfos
